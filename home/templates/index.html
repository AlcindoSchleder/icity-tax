Recursos da API

API i-City-tax tem a finalidade de provisionar aos nossos clientes uma forma fácil e rápida de calcular os impostos de venda dos produtos. Listamos abaixo alguns recursos da API i-City-tax:

Classificação NCM

A API recebe do cliente o código NCM do Produto e o estado de destino do da venda, que nos permite então, encontrar a alíquota certa para cada produto.

Informação em Lote

O cliente deve enviar uma lista dos produtos no formato pré-definido, assim retornamos as alíquotas para cada um dos produtos informados.

Retorno das Alíquotas

Depois de realizada a busca das Alíquotas, e, se informada a quantidade dos produtos, a API vai calcular o total do imposto para cada produto, o valor da Base de Cáculo, o valor da Redução da base de cálculo (se exisitir) e as mensagens informativas definidas por lei para os produtos da NF
